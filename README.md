# Mario Focus Timer

Una web app hecha con React + Vite para trabajar por rondas cortas con vibra tipo Mario:

- 40 segundos de preparación
- 3 iteraciones de trabajo
- 20 segundos de break entre rondas
- música retro generada con Web Audio API
- beep de transición
- jingle final victorioso
- confetti al completar

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Stack

- React
- Vite
- lucide-react
- Web Audio API

## Nota

No usa archivos de audio externos. Todo el sonido se sintetiza en el navegador.
